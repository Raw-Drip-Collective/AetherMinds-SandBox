# Contributing to AetherMinds SandBox

Thank you for your interest in contributing to AetherMinds SandBox!

## How to Contribute

### Reporting Bugs

1. Search existing issues to avoid duplicates
2. Create a new issue with:
   - Clear title describing the bug
   - Steps to reproduce
   - Expected vs actual behavior

### Pull Requests

1. Fork the repository
2. Create a feature branch:
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. Make your changes
4. Commit and push:
   ```bash
   git add .
   git commit -m 'Add: Your feature'
   git push origin your-branch-name
   ```
5. Open a Pull Request

## Development Setup

```bash
# Clone repository
git clone https://github.com/Raw-Drip-Collective/AetherMinds-SandBox.git
cd AetherMinds-SandBox
```

## Questions?

Open an issue for questions about contributing.
